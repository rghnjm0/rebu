## Текущие возможности

- Регистрация и авторизация пользователей
- Создание и просмотр постов
- Создание и просмотр сообществ
- Комментарии к постам
- Голосование за посты
- Простой адаптивный дизайн
- Главная страница + страницы отдельных сообществ

## Установка и запуск

### Требования

- Python 3.8 или выше
- Git для клонирования репозитория

### Шаги

1. Клонирование репозитория
```
git clone https://github.com/ВАШ_НИК/rebu.git
cd rebu
```
2. Создание виртуального окружения
```
python -m venv venv
venv\Scripts\activate
```
3. Установка зависимостей
```
pip install flask
```
4. Запуск приложения
```
python app.py
```
5. Проверка работоспособности

После запуска приложение будет доступно по адресу:

- http://127.0.0.1:5000

## Использование

- Зарегистрируйтесь → /auth/register
- Войдите → /auth/login
- Создайте сообщество → «создать сообщество»
- Создайте пост → «новый пост»
- Голосуйте и комментируйте
