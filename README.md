## Текущие возможности

- Регистрация и авторизация пользователей
- Создание и просмотр постов
- Создание и просмотр сообществ
- Комментарии к постам
- Голосование за посты
- Простой адаптивный дизайн
- Главная страница + страницы отдельных сообществ

## Использование

- Зарегистрируйтесь → /auth/register
- Войдите → /auth/login
- Создайте сообщество → «создать сообщество»
- Создайте пост → «новый пост»
- Голосуйте и комментируйте
